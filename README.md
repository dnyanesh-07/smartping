# SmartPing - API Testing Client for Visual Studio

**SmartPing** is a powerful, privacy-first API testing extension for **Visual Studio 2019/2022**. Test your APIs directly inside the IDE — no context switching, no external apps, no login required.

<img width="610" height="81" alt="location" src="https://github.com/user-attachments/assets/774af587-245c-4042-973b-3d1dbd828cf4" />


## 🚀 Accessing SmartPing
- Go to **Tools → SmartPing** after installation.  
- The interface opens as a dockable window — pin or dock it anywhere you prefer (recommended: bottom near *Output* window).

---

## 💡 Why SmartPing?
SmartPing is a **privacy-first alternative to Postman**, designed for developers who want **speed, simplicity, and security** — all within Visual Studio.

## 🧩 Core Features

### 🏠 Home Page – API Testing
- Full HTTP/REST support (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS)  
- Customize **headers, parameters, cookies, auth, and form-data**  
- Supports JSON, XML, and raw body formats  
- Real-time response viewer with syntax highlighting  

<img width="954" height="394" alt="runnerUI" src="https://github.com/user-attachments/assets/d90b5f6f-0a0d-428e-bcf3-bf434e07c8bc" />


### 🔁 Runner Page – Multiple Runs
- Hit the **same API multiple times** with **dynamic data changes**  
- Automate bulk testing or stress test endpoints easily  

### 🌐 Environment Management
- Create and manage multiple environments like **Dev**, **Staging**, and **Production**  
- Variable interpolation in URLs, headers, and request bodies

- **100% Local Storage** – your data never leaves your machine  
- **No Login Required** – start testing instantly

- **Zero Tracking** – no data analytics or cloud sync

## 🧭 Getting Started
1. Install **SmartPing** from Visual Studio Marketplace  
2. Open any solution in **VS 2019/2022**  
3. Access it via **Tools → SmartPing**  
4. Import or create your requests  
5. Start testing instantly — no signup, no setup
